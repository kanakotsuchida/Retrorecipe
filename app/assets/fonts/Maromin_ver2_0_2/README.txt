# このフォントについて
こちらはMaromin-Regular(2.0.2)およびMaromin-Mincho(2.0.2)です。ひらがなに対応しています。縦書きが可能です。
ご利用上問題が起こった場合、For_Errorフォルダ内のREADME.txtをお読みの上、そちらのフォントファイルをお試しください。
